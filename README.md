WELCOME
───────
Thank you for using AutoStrike 3.0.

AutoStrike is a lightweight Windows utility that lets you
automate mouse clicks and keyboard key presses with
precise timing control — no scripting knowledge required.


FEATURES
────────
  • Mouse auto-clicker with configurable interval,
    button, click type, and cursor position
  • Key presser supporting single keys and combinations
    (e.g. E, Ctrl+C, Alt+F4)
  • Global hotkeys to start/stop without switching windows
  • Customizable hotkey bindings
  • Settings saved automatically between sessions
  • Clean, no-bloat interface — no ads, no telemetry


INSTALLATION
────────────
  1. Run INSTALLER.bat as Administrator
  2. A shortcut will be placed on your Desktop and
     in your Start Menu under AutoStrike
  3. Launch AutoStrike from either shortcut


UNINSTALLATION
──────────────
  Option A: Control Panel → Apps → AutoStrike → Uninstall
  Option B: Start Menu → AutoStrike → Uninstall AutoStrike

  All files and registry entries are fully removed.
  No traces left behind.


USAGE NOTES
───────────
  • Some applications (games, system dialogs) may require
    AutoStrike to be run as Administrator to send input.
    Right-click the shortcut → Run as Administrator.

  • Windows Defender or antivirus software may flag
    AutoStrike on first launch. This is a known false
    positive common with compiled Python applications.
    The software contains no malicious code.
    You may need to click "More info → Run anyway" in
    Windows SmartScreen on first launch.

  • AutoStrike does not connect to the internet.
    It does not collect, transmit, or store any personal
    data of any kind.
